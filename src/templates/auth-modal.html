GitHub Authentication Setup

To create recipes, you need a GitHub Personal Access Token:

  1. Go to GitHub Settings → Personal access tokens → Tokens (classic)
  2. Click "Generate new token (classic)"
  3. Set a note (e.g., "Kuchtik Recipe App")
  4. Select scopes: repo and user
  5. Click "Generate token"
  6. Copy the token and paste it below
Security Note: The token will be stored locally in your browser. Never share your token with others.
